# 🎮 GameVault

**Your ultimate destination for premium PC games.**
Discover the latest releases, legendary titles, and exclusive deals. Fast, secure, and always evolving.

## 🚀 Tech Stack

- **Frontend**: Next.js (v16), Tailwind CSS, JSX
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Fonts**: Satoshi (Headings), Work Sans (Body)

## 🛠️ Features

- **Landing Page**: Dynamic Hero Section, Genre Filters, and Trends Grid.
- **Game Library**: Full searchable catalog with detailed game pages.
- **Authentication**: Secure user login and registration pages.
- **Wishlist**: Dedicated "Favorites" page for saved titles.
- **Deals**: "On Sale" section for exclusive discounts.
- **Community**: Integrated Blog for latest gaming articles.

## 📦 Installation

1. Clone the repo: `git clone https://github.com/asadnaqvi09/GameVault--nextjs.git`
2. Install dependencies: `npm install`
3. Set up your `.env` file (see .env.example)
4. Run development server: `npm run dev`
